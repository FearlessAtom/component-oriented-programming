## Prerequisites

Before running this project, ensure you have the following installed:

- **Node.js**: version `14.17.0` or higher
- **npm**: version `6.14.0` or higher

:::tip Version Check
You can verify your installed versions by running these commands in your terminal:
`node -v` and `npm -v`
:::

## Installation & Run

Execute these commands in order within your terminal:

1. **Clone the repository:**
```bash
git clone https://github.com/FearlessAtom/component-oriented-programming.git
```

2. **Enter the project directory**
```bash
cd component-oriented-programming
```

3. **Install all the required packages**
```bash
npm install
```

4. **Start the server**
```bash
npm run start
```
